Furniture Recommendation API

This API provides furniture recommendations using three different approaches:

  • Content-based filtering (similar items)
  • Collaborative filtering (user-based)
  • Hybrid recommendations (combining both approaches with style matching)

Endpoints

Content-Based Recommendations

Get recommendations for items similar to a specified item.

URL: /api/content/recommend/{item_id}

Method: GET

URL Parameters:

  • item_id - ID of the item to find similar items for

Query Parameters:

  • n - Number of recommendations to return (default: 5)

Example:

GET /api/content/recommend/item1?n=5

Collaborative Recommendations

Get personalized recommendations for a user based on their behavior.

URL: /api/collaborative/recommend/{user_id}

Method: GET

URL Parameters:

  • user_id - ID of the user to get recommendations for

Query Parameters:

  • n - Number of recommendations to return (default: 5)

Example:

GET /api/collaborative/recommend/user1?n=5

Hybrid Recommendations

Get recommendations that combine collaborative filtering, content-based filtering, and style matching.

URL: /api/hybrid/recommend/{user_id}/{item_id}

Method: GET

URL Parameters:

  • user_id - ID of the user
  • item_id - ID of the selected item

Query Parameters:

  • n - Number of recommendations to return (default: 4)

Example:

GET /api/hybrid/recommend/user1/item1?n=4
